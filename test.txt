	8 data types int,long,short,float,double,boolean,byte,character.
	int->store the integer values
	float-> store the decimal values
	boolean-> stores the values based on condition whether true or false
	long->stores the long value like phone number 10 digits and should be written L in last
	char-> used to store single value character.
	double-> used to store the values like price double price=345.6; .
	byte-> used to store values ranging from -126 to 126.
	short-> used to store short values.
	non-primitive data types:
	String->used to store the sequence of character. ex:static String name="Yogesh N";
	Array-> used to store multiple data values in single variable of same data type.
	List-> used to store multiple values of different data types in a single variable.
	
4) What is variables? Explain the types of variables?
-> Variable is a container for storing the data values. the values are stored in memory.
	Types of Variable:
	Local variable-> the variable which is declared inside the method is called local variable.
		Ex: class Rocket
			{
			public static void main(String [] args)
			{
			String rocketName="Rocket";
			System.out.println("Rocket Name:"+rocketName);
			}
			}
	Static variable-> the variable which is declared inside the class and outside the methods using 
						static keyword is called static variable.
		Ex: class Laptop
			{
			  Static string laptopName="Dell";
			  
			  public static void main(String [] lap)
			  {
			  System.out.println("Laptop Name:"+laptopName);
			  }
			}
	Instance Variable or Non Static variable-> the variable which is declared inside the class outside
												without using static keyword is called instance variable.
		Ex: class Television
			{
			String brandName="Sony";
			
			public void displayTvDetails()
			{
			System.out.println("Brand Name:"+brandName);
			}
		    }
			
			class TelevisionRunner
			{
			public static void main(string [] tv)
			{
			Television tv=new Television();
			tv.displayTvDetails();
			}}
			
	Parameterized variable-> the variable which is declared inside the parameter and initialized the
							value of variable in another method is called parameterized variable.
			class Television
			{
			public void displayTvDetails(String brandName)
			{
			System.out.println("Brand Name:"+brandName);
			}
		    }
			
			class TelevisionRunner
			{
			public static void main(string [] tv)
			{
			Television tv=new Television();
			tv.displayTvDetails("Sony");
			}}

6) What is Methods? Explain types of methods?
-> Method is a block of code or statements to perform a specific task. The main purpose of creating a
	method is to reuse the code and logics again and again.
	
	Types of Methods:
		Pre Defined Methods:
		- the  methods which is included already in the system is called pre defined methods.
		  static methods and instance methods
		  methods declared using static keyword is called static methods. 
			public static  void display()
		  methods declared without using static keyword is called instance methods.
			public void display()
			
		User defined methods:
		 The methods defined by users is called user defined methods.
		 ex: print(),println(),length()
